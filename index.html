Privacy Policy for Goathy The Chatbot

Last updated: January 2026

1. Data Collection and Processing

Goathy The Chatbot does not collect or store personal data by default. Messages are processed temporarily only to provide core functionality and are not stored permanently.

2. User-Initiated Contact with the Developer

The Bot provides a special command allowing users to send messages directly to the developer. By using this command, users consent to the message content and their Discord User ID being visible to the developer. This data is used solely for support purposes.

3. Data Retention

Messages sent through the contact command may be retained temporarily by the developer solely to respond to users. Data is not used for analytics, profiling, advertising, or tracking.

4. Data Sharing

The Bot does not sell, trade, or share user data with third parties. Data is only accessible to the developer when explicitly provided by the user through a consent-based command.

5. User Rights

Users may choose not to use the contact command if they do not wish to share their User ID or message. Users may request deletion of previously sent messages by contacting the developer directly.

6. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected on this page with an updated revision date.

7. Contact

If you have any questions regarding this Privacy Policy, you may contact the developer via Discord.

By using Goathy The Chatbot, you agree to this Privacy Policy.
